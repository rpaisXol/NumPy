# Python NumPy Study

This repository contains learning records for **NumPy**, the core scientific computing library for Python data analysis and machine learning. It is organized step-by-step, covering everything from creating N-dimensional arrays (ndarrays) to indexing, transformations, mathematical operations, and practical quizzes.

## Study Goals

- Understand the concept of multidimensional arrays (ndarrays) and freely manipulate data

- Learn computational speed optimization techniques through broadcasting and vectorization operations

- Master essential NumPy functions for data preprocessing, such as indexing, slicing, merging, and splitting


## Learning Objectives
- Enhance media file (image/video) control capabilities using OpenCV
- Understand and apply core image processing techniques (cropping, rotation, mirroring, transformation)
